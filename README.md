# URL-auto-opener
Automaticly opens a list of inputed URL(s). Written in Python 3.7

Requires webbrowser
use pip install webbrowser

Basic structure
  
  The "Starting URL"   The "Values"
  
                   |         |
                   
       ____________|______ __|___
       
        https://google.com/qwerty
